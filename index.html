GitHub Token

Для работы нужен персональный токен GitHub со scope repo.
Токен сохраняется только в вашем браузере и никуда не отправляется кроме api.github.com.

Создать токен: github.com → Settings → Developer settings → Personal access tokens → Tokens (classic)
Поставьте галочку repo → Generate token → скопируйте ghp_...
